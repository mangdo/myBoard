## 2. 사용 기술

`Back-end`
- Java 8
- SpringBoot 2.5.2
- Junit 5
- Gradle 7.0.2
- JPA
- MySQL 8.0

`Front-end`
- JQuery 3.5.1
- Bulma 0.9.2

`deploy`
- AWS EC2(Ubuntu 18.04 LTS)
- AWS RDS(MySQL 8.0)
